# AI 教材轉換器 - 使用說明

## 📖 簡介

AI 教材轉換器是一個專為教育訓練人員設計的工具，能夠將 Word 或 PDF 文件透過 AI 智慧分析，自動轉換為排版精美、結構清晰的專業教材。

## 🚀 快速開始

### 1. 開啟應用程式

雙擊 `index.html` 檔案，使用瀏覽器開啟（建議使用 Chrome 或 Edge）。

### 2. 四步驟製作教材

#### 步驟一：上傳檔案
- 點擊上傳區域或拖曳檔案
- 支援格式：Word (.docx) 或 PDF
- 檔案中的圖片會自動保留

#### 步驟二：AI 智慧分析
- 點擊「🪄 開始 AI 分析」按鈕
- AI 會自動：
  - 標記重點內容
  - 定義專業術語
  - 建議圖片插入位置

#### 步驟三：樣式設定
- 選擇班級主色調（6 種預設顏色）
- 或使用自訂顏色選擇器
- 即時預覽效果

#### 步驟四：下載教材
1. 點擊「🔄 計算頁碼」（重要！）
2. 選擇下載格式：
   - 📄 下載 PDF：適合列印和分發
   - 📝 下載 Word：方便後續編輯

## 💡 進階功能

### 專案管理

#### 儲存草稿
- 點擊「儲存草稿」按鈕
- 專案會儲存在瀏覽器中
- 可隨時繼續編輯

#### 載入專案
- 點擊「載入專案」按鈕
- 選擇要繼續編輯的專案
- 支援刪除不需要的專案

### 預覽功能

#### 縮放
- 使用 + / - 按鈕調整預覽大小
- 方便檢視細節

### API 設定

#### 更換 API Key
- 點擊「更換 API Key」按鈕
- 輸入您的 Google Gemini API Key
- 系統會記住您的設定

## ⚠️ 重要提醒

### 計算頁碼
**必須在下載 PDF 前執行！**

網頁是連續捲動的，而 PDF 是分頁的。系統需要透過「計算頁碼」功能模擬印刷後的分頁位置，才能讓目錄頁碼正確對應。

### 下載 PDF 設定

當瀏覽器跳出列印視窗時，請確認以下設定：

1. **目的地**：選擇「另存為 PDF」
2. **紙張尺寸**：A4
3. **邊界**：預設或無
4. **背景圖形**：✅ **務必勾選！**（否則顏色和圖片會消失）

## 🎨 顏色主題

系統提供 6 種預設班級顏色：

- 🔵 藍色班級
- 🔴 紅色班級
- 🟢 綠色班級
- 🟣 紫色班級
- 🟠 橘色班級
- 🔷 青色班級

您也可以使用自訂顏色選擇器設定任何顏色。

## 📝 支援的內容類型

系統會自動識別並美化以下內容：

- **章節標題**：大標題，會自動分頁
- **小節標題**：次標題
- **一般段落**：內文
- **重點提示**：藍色框框
- **術語定義**：綠色框框
- **警告事項**：紅色框框
- **圖片**：自動排版
- **表格**：自動格式化

## ❓ 常見問題

### Q1：為什麼圖片或顏色在 PDF 中消失了？
**A**：請確認列印視窗中有勾選「背景圖形」選項。

### Q2：目錄頁碼不正確怎麼辦？
**A**：在下載前重新點擊「🔄 計算頁碼」按鈕。

### Q3：可以上傳多大的檔案？
**A**：最大支援 10MB 的檔案。

### Q4：支援哪些瀏覽器？
**A**：建議使用 Chrome、Edge 或 Firefox 最新版本。

### Q5：資料會上傳到雲端嗎？
**A**：不會！所有資料都儲存在您的瀏覽器中，完全離線運作（除了 AI 分析需要連網）。

### Q6：如何取得 Google Gemini API Key？
**A**：
1. 前往 https://aistudio.google.com/
2. 登入 Google 帳號
3. 點選「Get API Key」
4. 創建並複製 API Key

## 🔧 技術支援

如遇到問題，請檢查：

1. 瀏覽器版本是否為最新
2. 網路連線是否正常（AI 分析需要）
3. API Key 是否正確設定
4. 檔案格式是否為 .docx 或 .pdf

## 📚 使用技巧

### 最佳實踐

1. **檔案準備**：
   - 使用清晰的標題結構（H1、H2、H3）
   - 適當使用粗體標記重點
   - 圖片品質要清晰

2. **AI 分析**：
   - 內容越結構化，AI 分析效果越好
   - 專業術語會被自動識別

3. **顏色選擇**：
   - 不同班級使用不同顏色便於區分
   - 建議使用預設顏色，視覺效果較佳

4. **定期儲存**：
   - 編輯過程中定期點擊「儲存草稿」
   - 避免瀏覽器關閉導致資料遺失
