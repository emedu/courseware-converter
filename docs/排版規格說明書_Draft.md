# 教材排版規格說明書 (Draft)

這份文件用於定義教材轉換系統的最終輸出樣式。請檢視以下各項設定，並標註您的偏好。

## 1. 頁面佈局 (Page Layout)

- **紙張大小**: A4 (21cm x 29.7cm) [固定]
- **頁面方向**: 直向 (Portrait) [固定]
- **頁邊距 (Margins)**:
  - [ ] **選項 A (標準/列印)**: 上下左右皆 2.54cm (Word 預設)
  - [x] **選項 B (緊湊/閱讀)**: 上下 2cm, 左右 2cm (內容更多)
  - [ ] **選項 C (寬鬆/筆記)**: 左右 3.17cm (留白多，方便筆記)

## 2. 字體與文字 (Typography)

### 內文 (Body Text)
- **中文字體**:
  - [ ] 新細明體 (傳統計公文感)
  - [x] 微軟正黑體 (現代螢幕閱讀感，推薦)
  - [ ] 標楷體 (正式教育文件感)
- **字號 (Size)**:
  - [ ] 10.5pt (緊湊)
  - [x] 12pt (標準閱讀)
  - [ ] 14pt (長輩/兒童友善)
- **行距 (Line Spacing)**:
  - [ ] 1.0 倍 (單行間距，最緊)
  - [x] 1.15 倍 (標準，微軟正黑體建議)
  - [ ] 1.5 倍 (寬鬆，適合批改)

### 標題 (Headings)
*標題應具備「黏著性」(Keep with next)，確保不與下文分頁。*

- **Level 1 (大章節標題)**:
  - 樣式: [x] 置中加粗 / [ ] 靠左加粗
  - 字號: 18pt
  - 換頁: [x] 強制換頁 (每一章從新頁開始) / [ ] 接續上一章

- **Level 2 (小節標題)**:
  - 樣式: [x] 靠左加粗
  - 字號: 16pt
  - 裝飾: [ ] 無 / [x] 下方加細灰線 / [ ] 左側深色色塊

- **Level 3 (子標題)**:
  - 樣式: 靠左加粗
  - 字號: 14pt

## 3. 圖片與圖說 (Images)

- **圖片寬度**:
  - [ ] 100% 頁寬 (滿版)
  - [x] 80% 頁寬 (置中，兩側留白)
  - [ ] 原圖尺寸 (不做縮放)
- **佔位符 (Placeholder)**: *當沒有圖片時顯示的框*
  - 高度: [x] 固定 8cm (約佔 1/3 頁) / [ ] 固定 5cm / [ ] 彈性
  - 樣式: 灰色背景 + 虛線邊框 + "請插入圖片" 文字
- **圖說 (Caption)**: 圖片下方的說明文字
  - [x] 顯示 (置中，灰色，斜體)
  - [ ] 不顯示

## 4. 表格 (Tables)

- **寬度**: 100% 頁寬
- **表頭 (Header)**:
  - [ ] 深色背景 + 白字 (強烈對比)
  - [x] 淺灰背景 + 黑字 (柔和，省墨水)
  - [ ] 無背景 + 粗體
- **格線**: 
  - [x] 全格線 (標準)
  - [ ] 僅橫線 (簡約)

## 5. 頁首頁尾 (Header & Footer)

- **頁首**:
  - 左側: [x] 教材名稱 / [ ] 空白
  - 右側: [ ] 章節名稱 / [x] 日期 / [ ] 空白
- **頁尾**:
  - 置中: `第 X 頁` 或是 `第 X 頁 / 共 Y 頁`
