# 教材排版規格說明書 (職訓教材版)

本規格書依照「專業及職業訓練教材」之編撰準則制定。
**核心設計理念**：結構化 (Structured)、易讀性 (Readable)、操作導向 (Action-Oriented)。

## 1. 頁面佈局 (Layout Context)

職訓教材通常配合講義夾或實務操作，需要較大的邊距做筆記與翻閱。

- **紙張**: A4 直向
- **頁邊距 (Margins)**: **寬鬆配置**
  - 上: 2.54 cm
  - 下: 2.54 cm
  - 左: **3.0 cm** (預留裝訂孔位/筆記區)
  - 右: 2.5 cm

## 2. 字體策略 (Typography Strategy)

採用「黑體主導」策略，提升現代感與數位閱讀體驗，並符合職訓教材的俐落風格。

- **內文 (Body)**:
  - 字體: **微軟正黑體 (Microsoft JhengHei)**
  - 字號: **12pt** (標準)
  - 行距: **1.25倍** (較寬鬆，利於閱讀長文)
  - 段落間距: 段後 0.5 行 (區塊分明)

- **標題層級 (Visual Hierarchy)**:
  - **Level 1 (章名)**:
    - 字號: **20pt**
    - 樣式: **加粗 + 獨立一頁**
    - 裝飾: 標題下方加 **雙粗線** 或 **色塊底色**
  - **Level 2 (節名)**:
    - 字號: **16pt**
    - 樣式: **加粗**
    - 裝飾: 標題下方加 **單細線** (貫穿頁寬)
  - **Level 3 (小標)**:
    - 字號: **14pt**
    - 樣式: **加粗 + 左側豎線** ( | 標題)

## 3. 視覺輔助系統 (Visual Aids)

職訓教材強調「重點在哪裡」，因此不使用純文字描述重點，而使用區塊。

- **重點/提示框 (Keypoint Box)**:
  - 樣式: **淺藍/淺灰底色 + 深色邊框**
  - 圖示: 左上角加入 `💡` (提示) 或 `⚠️` (注意)
  - 縮排: 左右各縮 1cm (比內文窄，凸顯區塊感)

- **名詞定義 (Definition)**:
  - 樣式: **粗體專有名詞 + 冒號 + 一般解釋**
  - 例如：**ODT技術**：一種利用密封....

## 4. 圖片規範 (Image Standards)

- **位置**: 一律 **水平置中**
- **尺寸**:
  - 寬度: 頁寬的 **80% ~ 90%** (確保細節清晰)
  - 高度: 依比例自動，但在 Word 中預設 **不鎖定錨點** (In line with text)，避免跑版。
- **圖說 (Caption)**:
  - 格式: `圖 [章]-[序]：[說明]` (例如：圖 2-1：粉刺夾持手勢)
  - 樣式: 置中、10pt、深灰色

## 5. 頁眉頁腳 (Navigation)

- **頁眉**:
  - 左: `[教材名稱]`
  - 右: `第 [X] 章 [章節名稱]` (讓學員知道現在讀到哪)
  - 分隔線: 下方加一條實線

- **頁腳**:
  - 置中: `Page - [頁碼] -`
